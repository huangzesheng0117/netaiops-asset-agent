# NetAIOps 网络信息统一查询入口 V1 使用说明

## 1. 支持能力

V1 已支持：

1. 通过自然语言查询 CMDB 网络设备资产。
2. 通过管理 IP、主机名、序列号、EM 码等字段反查其他字段。
3. 查询 IDC、机房、机柜、环境、厂商、型号、状态等条件。
4. 查询结果以表格方式返回。
5. 查询结果导出 Excel。
6. 支持对话历史。
7. 支持“导出刚才结果 Excel”这类对话动作。

## 2. 查询示例

- 10.189.250.8 是哪台设备，主机名、序列号、型号、状态、IDC、机房、机架是什么？
- 设备序列号为 FDO24130P9S 的设备，主机名和管理 IP 分别是多少？
- EM码为 EM06027 的设备主机名是什么？
- SH8 机房 G 排机柜，生产网的设备有哪些？
- WG88-SW-H19-1 这台设备的用途是什么？操作系统是什么？

## 3. 导出 Excel

完成一次查询后，可以点击页面中的“导出结果 Excel”。

也可以继续输入：

- 导出刚才结果 Excel

平台会基于上一轮查询条件生成 Excel 下载链接。

## 4. 注意事项

1. 当前 V1 只查询网络设备资产，对应 CMDB 的 networkServer 接口。
2. 当前 V1 不做任何配置修改。
3. 单次页面默认返回 20 条，可在输入框右下角调整返回条数。
4. 单次 Excel 最多导出 500 条。
5. 如果 CMDB 数据本身不完整，平台不会编造不存在的数据。
