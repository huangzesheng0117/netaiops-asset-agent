# NetAIOps 网络信息统一查询入口 V1 回滚说明

## 1. 常规回滚原则

每批变更脚本都会在 /opt/netaiops-asset-agent/backup/ 下生成备份目录，例如：

- /opt/netaiops-asset-agent/backup/batch27_xxx
- /opt/netaiops-asset-agent/backup/batch28_xxx

如需回滚某批变更，优先使用对应备份目录中的文件恢复。

## 2. 示例：恢复 app.py

    cp -a /opt/netaiops-asset-agent/backup/<backup_dir>/app.py.bak /opt/netaiops-asset-agent/app.py
    chown netaiops:netaiops /opt/netaiops-asset-agent/app.py
    chmod 640 /opt/netaiops-asset-agent/app.py
    systemctl restart netaiops-asset-agent

## 3. 示例：恢复配置文件

    cp -a /opt/netaiops-asset-agent/backup/<backup_dir>/config.yaml.bak /etc/netaiops-asset-agent/config.yaml
    chown root:netaiops /etc/netaiops-asset-agent/config.yaml
    chmod 640 /etc/netaiops-asset-agent/config.yaml
    systemctl restart netaiops-asset-agent

## 4. 回滚后验证

    cd /opt/netaiops-asset-agent
    source venv/bin/activate
    ./tools/v1_status_check.sh
    ./tools/v1_acceptance_check.sh
